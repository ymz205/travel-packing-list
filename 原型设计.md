# 一、原型设计核心原则

遵循 PRD 中「极简主义、3 步快速生成、进度可视化、大按钮易操作」的核心要求，整体设计：

- 视觉风格：极简白为主，品牌色（建议 #4285F4）点缀，无冗余装饰；
- 交互逻辑：3 步核心流程闭环，操作反馈即时可见；
- 响应式：适配移动端（375px-414px）、平板（768px）、桌面端（1200px）。

# 三、关键交互动效（高保真核心）

1. **清单生成动效**：点击「生成清单」后，页面加载时显示骨架屏（列表项灰色占位），200ms 后逐步渲染物品列表（从分类头部开始，每项淡入 + 轻微上移）；
2. **勾选反馈**：勾选复选框时，物品文字 0.3s 内完成划线 + 变灰，顶部进度条 / 环形进度实时更新（数字 + 进度动画）；
3. **极简模式切换**：开启极简模式时，列表项按重要性筛选，低优先级物品 0.5s 内淡出移除，剩余物品数量实时更新；
4. **撤销 / 重做**：Ctrl+Z/Ctrl+Y 时，页面右上角显示「撤销成功 / 重做成功」Toast（3s 自动消失，无遮挡核心区域）；
5. **响应式适配**：移动端列表项取消备注 / 删除按钮的 hover 显示，改为点击列表项右侧「⋮」展开操作菜单；桌面端保留 hover 交互。

# 四、视觉规范补充（高保真落地关键）

| 元素类型 | 规范要求                                                     |
| -------- | ------------------------------------------------------------ |
| 字体     | 全局使用无衬线字体（移动端：SF Pro / 思源黑体；桌面端：Inter / 微软雅黑）；标题：18-20px/bold；正文：16px/regular；辅助文字：14px/light |
| 颜色     | 主色：#4285F4（品牌蓝）；辅助色：#EA4335（删除 / 警告）、#34A853（成功）；中性色：#FFFFFF（背景）、#F5F5F5（列表项 hover）、#9E9E9E（辅助文字）、#212121（正文） |
| 圆角     | 全局统一圆角：按钮 / 输入框 8px；模态框 / 卡片 12px；复选框 / 开关全圆角 |
| 间距     | 基础间距单位 8px，所有控件间距为 8px 的倍数（8/16/24/32）；移动端页面左右边距 16px，桌面端 24px |
