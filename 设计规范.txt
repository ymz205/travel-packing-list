旅行类APP设计规范文档
一、设计风格
整体风格定位：自然清新 · 现代简约 · 温暖治愈
表格
维度	设计特征
视觉基调	以自然风景为核心视觉元素，大量运用高质量实景摄影，营造沉浸式旅行氛围
设计语言	现代简约（Modern Minimalism）+ 新拟态（Soft UI）结合，卡片式布局为主
情感传达	温暖、放松、探索感，通过柔和圆角、自然色调传递治愈感
空间层次	大量留白 + 半透明毛玻璃效果（Glassmorphism），提升内容呼吸感
交互风格	轻量、流畅，底部Tab导航 + 浮动操作按钮（FAB）

二、色彩系统
2.1 主色（Primary Colors）
表格
色彩角色	色值	使用场景
品牌主色-深绿	#1B4D3E 或 #2D5A4A	导航栏、重要按钮、品牌标识、核心标签
品牌主色-浅绿	#7CB342 或 #8BC34A	次要按钮、选中状态、图标高亮、价格标签
背景色-米白	#F5F5F0 或 #FAFAF5	页面主背景、卡片底色
设计说明：深绿色传递自然、探索、可信赖的品牌联想；浅绿色作为活力点缀，打破沉闷；米白色背景替代纯白，降低视觉疲劳，提升温暖感。

2.5 色彩应用模式
plain
页面层级配色：
├─ 顶部导航区：深绿(#1B4D3E) 或 透明 + 深色文字
├─ 首屏Banner：全屏图片 + 底部渐变遮罩(黑→透明)
├─ 功能入口区：浅绿图标 + 深绿文字，白色卡片背景
├─ 内容卡片区：米白背景(#F5F5F0) + 白色卡片 + 浅灰边框
├─ 底部导航区：白色背景 + 深绿选中态 + 灰色未选中态
└─ 浮动按钮：浅绿(#7CB342) 或 深绿，白色图标

3.4 排版原则
亲密性：相关信息间距 8px，模块间距 24-32px
对比度：正文与背景对比度 ≥ 4.5:1（符合WCAG AA标准）
对齐：统一左对齐，价格/数字右对齐
行宽：单行文字不超过 35 个中文字符，避免阅读疲劳

4.3 图标风格
风格：线性图标（2px描边）+ 面性图标结合
尺寸：Tab栏 24×24px，功能入口 40×40px，详情页 20×20px
色彩：默认灰色(#9E9E9E)，选中深绿(#1B4D3E)或浅绿(#7CB342)
圆角：图标圆角 2-4px，保持视觉柔和
五、设计原则总结
"自然为底，内容为王，细节见品质"
图片优先：旅行类产品以高质量图片为核心，文字作为辅助说明
色彩克制：主色不超过3种，用中性色承载大部分信息
留白呼吸：模块间距 ≥ 24px，卡片内边距 16-20px
一致体验：跨页面保持色彩、字体、圆角、阴影的统一性
情感连接：通过暖色调和柔和形态，建立用户对品牌的信任与向往
